# Phishing Email Detection & Awareness System

## Author
Thanvesh Reddy  
B.Tech – SRM University  

## Objective
This project focuses on identifying phishing emails using manual analysis techniques and creating awareness to help users protect themselves from email-based cyber-attacks.

## Tools Used
- Public phishing email datasets
- Manual email content analysis
- Browser inspection
- Google Docs / MS Word
- GitHub

## Key Features
- Identification of phishing indicators
- Risk classification of emails
- Phishing awareness and prevention guide
- Business impact analysis

## Project Structure
- samples/ – Email samples
- analysis/ – Email analysis report
- awareness/ – Awareness guide
- report/ – Final PDF report

## Learning Outcomes
- Practical phishing detection skills
- Understanding of real-world cyber threats
- Security documentation experience
